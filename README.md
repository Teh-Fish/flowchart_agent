## This project create an agent specifically for creating a flowchart in draw.io
A LangGraph-based agent that converts natural language process descriptions into draw.io flowchart diagrams automatically.
### How to use:
 1. First create a `.env` file following the `config.env.example` format, note that the name of the file must be `config.env`
 2. Run `model.py` and use the terminal to interact with the llm
 3. If the create_flowchart_tool is called, a `diagram.drawio` file will be created in an `outputs` subdirectories
